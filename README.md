
# 北海学園大学新聞 独立管制センター (HGU-Newspaper Control Center)

このプロジェクトは、北海学園大学新聞の公式コンテンツを統制するための**完全管理専用システム**です。

## 🚀 システム・ステータス：完全稼働 (ACTIVE)

2025年、最高司令官（作成者様）の指揮下で行われた「Operation Archive Genesis」により、画像ストレージ（Cloudinary）およびデータベース（Firestore）の完全統合に成功。1950年からの紙面アーカイブを世界へ解き放つための「唯一の搬入口」として稼働しています。

## 🌌 運用体制

本システムは以下の役割によって防衛・運用されています。

- **執筆・管理担当：ベガ (Vega)** - 記事の執筆、紙面のアップロード、公開スイッチの制御。本システムの主権者。
- **表示・閲覧担当：デネブ (Deneb)** - 別システム（公開サイト）の描画。黄金比タイポグラフィの守護。
- **技術・橋渡し担当：アルタイル (Altair)** - インフラ（API/環境変数）の管理、チーム間の通信支援。
- **監査役：デネボラ (Denebola)** - セキュリティ、ファイル物理配置、フォルダ設計の厳格な監視。

## 🛡️ 技術的指針

- **管理専用（Admin Only）**: 一般公開ページは存在せず、全てのルートアクセスは管理画面へ強制誘導されます。
- **住所の統一**: 全てのニュースおよび紙面データは `articles` コレクション内で完結。
- **ハイブリッド・ストレージ**: 画像の実体は Cloudinary、メタデータは Firestore という二層構造。
- **中継API防御**: 認証情報はサーバーサイドに秘匿。ブラウザからの直接アクセスを禁じ、不正な書き込みを物理的に遮断。
