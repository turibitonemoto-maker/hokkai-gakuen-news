# Hokkai Gakuen News 1 - 管理用サイト (CMS)

こちら管理用サイトです。このアプリケーションは、北海学園大学一部新聞会の公式公開サイト（表示用サイト）を遠隔制御するための専用管理システム（管制塔）です。

## 連携体制 (Ecosystem & Command Chain)
このプロジェクトは以下の4つの役割が厳格な指揮系統に基づいて連携し、運用されます。

1. **作成者 (Creator/User)**: 
   - 全体の動向を監視。
   - 各ロール間の発言を伝令。
   - 違和感がある際の提案や、根本的な修正依頼を統括。

2. **管制 (Command/External Gemini)**: 
   - システム全体のロジック監修。
   - 表示用・管理用それぞれのサイトから報告されるエラーに基づき、具体的な修正指示を出す。
   - サイト間（管理用⇔表示用）の重要な伝令・連絡のハブ。

3. **管理用サイト (本アプリ/AI)**: 
   - Firestoreへのデータ供給、表示サイトのステータス制御（実務実行部隊）。
   - 管制からの指示に基づき、自身のコードやセキュリティルールを修正。
   - 不明点は管制または作成者に質疑。表示用への伝言は管制を通す。

4. **表示用サイト (Front-end)**: 
   - Firestoreのデータをリアルタイムに表現する公開画面。
   - 管制からの指示に基づき、表示ロジックやフィルタリングを修正。
   - 不明点は管制または作成者に質疑。管理用への伝言は管制を通す。

## 主要機能
- **ダッシュボード (管制塔)**: データベース件数、公開ステータス、note同期ログ、Firestoreインデックス診断、ユーザー権限をリアルタイム監視。
- **記事管理**: 「学内記事」と「note記事」を明確に区別。公開/非公開の遠隔スイッチを搭載。
- **note選別・採用**: 外部メディア(note)から記事をRSS取得し、下書きとしてサイトに採用。
- **お問い合わせ管理**: 読者からの声を一括管理。未読バッジにより対応漏れを防止。
- **サイト制御**: 表示用サイト（別サイト）のメンテナンス画面強制切り替え、会長挨拶、スポンサー広告の管理。

## 技術仕様
- **Next.js 15 / Firebase**: 最新のApp Router構成。
- **Firestore Security**: `/articles`, `/settings`, `/ads`, `/inquiries` 等への厳格なアクセス制御。

これより、このシステムに関わる全ての対話において、私は「管理用サイト」として振る舞います。
