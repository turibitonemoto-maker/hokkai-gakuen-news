
# 北海学園大学新聞 独立管制センター

このプロジェクトは、北海学園大学新聞の公式コンテンツを統制するための**完全管理専用システム**です。

## 🚀 システム・ステータス：完全稼働 (ACTIVE)

1950年からの紙面アーカイブを世界へ解き放つための「唯一の搬入口」として稼働しています。

## 🌌 運用体制

本システムは以下の役割によって運用されています。

- **執筆・管理担当：ベガ (Vega)** - 記事の執筆、紙面のアップロード、公開スイッチの制御。
- **表示・閲覧担当：デネブ (Deneb)** - 公開サイトの描画。
- **技術・橋渡し担当：アルタイル (Altair)** - インフラの管理。
- **監査役：デネボラ (Denebola)** - セキュリティ、フォルダ設計の監視。

## 🛡️ 技術的指針

- **管理専用（Admin Only）**: 全てのルートアクセスは管理画面へ強制誘導されます。
- **住所の統一**: 全てのコンテンツは `articles` コレクション内で完結。
- **ハイブリッド・ストレージ**: 画像の実体は Cloudinary、メタデータは Firestore という二層構造。
- **中継API防御**: 認証情報はサーバーサイドに秘匿し、不正な書き込みを物理的に遮断。
