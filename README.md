
# 北海学園新聞会 - 管理用サイト (CMS)

このアプリケーションは、北海学園大学一部新聞会の公式公開サイトを管理するための専用システムです。

## 連携体制
このプロジェクトは以下の4つの役割によって運用されます。

1. **作成者 (User)**: 全体の監修および運用。
2. **管制 (Command)**: ロジック監修および指示。
3. **管理用サイト (本アプリ)**: Firestoreへのデータ供給、表示サイトの制御。
4. **表示用サイト (Public)**: 公開画面。管理用が供給するデータの仕様を厳守。

## メンテナンス制御
表示用サイトの状況は以下のパスで制御可能です。
- **パス**: `/settings/maintenance`
- **フィールド**: `isMaintenanceMode` (boolean), `maintenanceMessage` (string)

## データ仕様
表示用サイトは、以下の条件でデータを取得してください。
- `articles`: `isPublished == true`
- `ads`: 有効期間内かつ `startDate` 以降
