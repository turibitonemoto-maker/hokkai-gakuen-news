
# 北海学園大学一部新聞会 管理システム

このプロジェクトは、北海学園大学一部新聞会の公式ニュース配信における**管理用サイト**です。

## プロジェクトの役割

本システムは、以下の「夏の大三角形」＋「監査役」体制によって運営・防衛されています。

- **執筆・管理担当：ベガ (Vega)** - 記事の執筆、編集、公開スイッチの制御。
- **表示・閲覧担当：デネブ (Deneb)** - 公開サイトの描画、黄金比タイポグラフィの守護。
- **技術・橋渡し担当：アルタイル (Altair)** - インフラ構築、環境変数の管理、チーム間の通信支援。
- **監査役：デネボラ (Denebola)** - セキュリティ、物理配置、フォルダ設計の厳格な監視。

## 技術的聖典 (Sacred Scripture)

- **完全自前主義（Independence）**: 外部プラットフォームへの依存を廃止。
- **住所の統一**: 全てのニュースおよび紙面データは `articles` コレクション内で完結。紙面データは `categoryId: "Viewer"` を付与。
- **報道品質タイポグラフィ**: 日本仕様の黄金比（行間1.6倍）を全域で適用。
- **ハイブリッド・ストレージ**: 画像の実体は Cloudinary、メタデータは Firestore という二層構造。
- **Digital Paper Viewer**: 複製リスクを抑えたセキュアな紙面配信。
